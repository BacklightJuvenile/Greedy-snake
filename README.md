# Greedy-snake
### 项目描述

这是一个基于html、css、JavaScript开发的贪吃蛇小游戏

### 项目目录

| 序号 | 名称      | 类型       | 作用               |
| ---- | --------- | ---------- | ------------------ |
| 1    | 外挂包    | 文件夹     | 开挂用的呗         |
| 2    | 文件汇总  | 文件夹     | 汇总成一个html文件 |
| 3    | main.html | html       | 主文件             |
| 4    | main.css  | css        | 渲染界面           |
| 5    | main.js   | javascript | 控制移动           |
